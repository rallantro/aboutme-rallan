ReciclaPet: A horta Inteligente - Medalha de Ouro

ReciclaPet: A horta Inteligente é um projeto que foi submetido para a AcelorMittal, meio-ambiente, onde ganhamos a medalha de ouro no estado de São Paulo e o 3° Lugar no Brasil.

O Projeto consistia em uma horta vertical integrada a um filto que filtava a água da chuva, cujo a irrigação era controlada por uma arduino.

Neste projeto eu fiquei respnsável pela programação da arduino e toda a parte de eletrônica.

Projeto LED Bluetooth

Este projeto eu submeti, como projeto final para o curso de programação física, onde uma arduino estava conectada a um led e um módulo HC-05, que permitia que ele se conectasse via bluetooth a um celular, que, através de um aplicatvo, bem simples, desenvolvido por mim, ultilisando o appInventor2.
